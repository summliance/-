# 💑 情侣点菜系统

一款专为情侣设计的点菜管理应用，帮助两个人更好地规划每日菜单、记录饮食历史、统计菜品偏好。

## ✨ 功能特性

### 🍽️ 菜品管理
- **自由录入**：添加菜品名称、分类、食材、做法等信息
- **个性化分类**：自定义菜谱分类，菜品可归类到多个分类
- **快速编辑**：随时查看、编辑或删除菜品
- **收藏功能**：收藏常用菜品，快速访问

### 📝 点菜与历史
- **一键点菜**：点击菜品卡片即可选中，底部浮窗实时显示已选菜品
- **历史记录**：按日期合并显示历史订单，统计每天的订单次数和菜品数量
- **灵活筛选**：按日期或菜品名称筛选历史订单
- **订单管理**：支持删除历史订单

### 📊 数据统计
- **多维度统计**：按日/周/月/总维度查看点菜频次
- **可视化图表**：直观的柱状图展示菜品排行
- **智能分析**：快速了解最受欢迎的菜品

### 🔍 搜索与分享
- **全文搜索**：按菜品名称、食材、分类快速搜索
- **分享链接**：生成分享链接，邀请好友查看菜单

### 💾 数据管理
- **数据导出**：将所有数据导出为JSON文件备份
- **数据导入**：
  - **合并导入**：保留现有数据，仅添加新数据
  - **替换导入**：清空现有数据后导入
- **清空数据**：一键清空所有数据（需二次确认）

## 🚀 快速开始

### 在线使用

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

### 本地运行

1. 克隆或下载本项目
2. 使用浏览器打开 `index.html`
3. 开始使用！

### 部署到 Vercel

1. Fork 本项目到你的 GitHub 账号
2. 在 [Vercel](https://vercel.com) 中导入项目
3. 点击部署，即可获得在线访问链接

## 💡 使用技巧

### 数据备份
建议定期使用"数据管理"功能导出数据，防止浏览器清理导致数据丢失。

### 跨设备同步
1. 在设备A导出数据
2. 将JSON文件传输到设备B
3. 在设备B使用"合并导入"或"替换导入"

### 两人共用
- 方式1：共用一个设备/浏览器
- 方式2：定期通过导出/导入功能同步数据
